Grants / Fund Applications Management using Odoo CRM

Grants Applications Management System in Odoo using CRM

Manage Grants as Grant Seeker and Grant Makers

This app allows you to have management of your grants / fund as grant seekers as well as grant makers.

Main Features:

  • Allow you to configure and manage grants applications as grant seeker and grant maker both.
  • Allow you to configure and manage grant application stages.
  • Allow you to configure and manage the application team.
  • Allow you to configure and manage the project team.
  • Allow you to configure and manage grant types.
  • Allow you to configure and manage grant methods.
  • Allow you to configure and manage grant tags.
  • Allow you to prepare proposals as grant seekers and allow you to create grant seeker applications from it. You can write a grant proposal and send it to your potential grantor or client.
  • Allow you to create / receive grant applications from your clients, charities (known as "grantseekers") .
  • Allow you to create a project from grant applications.
  • Links of grant application and grant requests on sales order, purchase order and invoice as shown in below screenshots.
  • Internal and external communication using open chatter of Odoo on Grant Application.
  • For more details please check below screenshots and watch the video.
