Remittance Advice Report

Overview

This App Generate to Remittance Advice in Vendor Payment and User able to set 'Remittance Advice: Send by email' template in 'Send receipt by email' popup.

Features

  • Users will print Remittance Advice in Vendor Payment.
  • Company Signature Imagewill show in Remittance Advice report from Company Information.
  • Remittance Advice: Send by email template set directly in 'Send receipt by email' popup, When configuration is enabled at popup.
  • Users can select Remittance Advice: Send by email template in 'Send receipt by email' popup when configuration is enabled at popup.
  • If configuration is not enabled at popup then only default odoo payment template will be visible.
  • Configuration field visible only in in vendor payment.
